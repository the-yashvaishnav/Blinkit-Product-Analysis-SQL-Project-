# 📊 Blinkit Product Analysis SQL Project

## 📌 Project Description
This project analyzes Blinkit sales data using SQL to extract insights like sales trends, customer behavior, and product performance.

---

## 🎯 Objectives
- Data analysis through SQL queries  
- Identify sales trends  
- Understand customer purchasing patterns  
- Generate business insights  

---

## 🛠️ Tools & Technologies
- SQL (MySQL)  
- MySQL Workbench / DBeaver (Database Tool)  
- GitHub  

---

## 📂 Dataset Description
- Database Name: ft_p  
- Table Name: blinkit  

---

## 📈 Key Insights
- The dataset contains a total number of item records, indicating the volume of data analyzed.  
- The maximum Item MRP shows the highest-priced product in the dataset.  
- The analysis shows the maximum item outlet sales for each outlet type, highlighting the best-performing outlet categories.
